Reporting bugs
==============

It's simple.

* Go to https://bugs.launchpad.net/fsaecostreport
* Click on *Report a bug*
* You will need to create an account on LaunchPad. 
  After that sign in with that account to continue.
* Enter a small summary of the bug and click *Next*
* In the *Further information* text box, enter more information about the
  bug.
  E.g. copy/paste the error obtained from the command line prompt.
* Wait for the bug to be acknowledged. 
  You will get an email about when the bug is fixed.
