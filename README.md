# fsaecostreport

![GitHub Workflow Status](https://img.shields.io/github/workflow/status/ppinard/fsaecostreport/CI)

Generator of the FSAE Cost Report


## Installation

For development installation from the git repository::

```
git clone git@github.com/ppinard/fsaecostreport.git
cd fsaecostreport
pip install -e .
```
